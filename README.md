# Foodie - An AI Powered Recipe Book

Foodie is an AI powered recipe book application designed to help users explore cuisines from different parts of the world through an interactive and aesthetic experience. The platform combines recipe discovery with conversational AI to create a space where cooking feels engaging, accessible, and creative.

Users can sign up or log in to access the platform and browse recipes inspired by global cuisines. Alongside recipe exploration, Foodie introduces Rey, an AI assistant integrated into the application to help users with cooking doubts, recipe understanding, ingredient questions, and general food related guidance.

The project follows a soft pixel inspired visual style with a pastel themed interface that focuses on creating a warm and visually pleasing user experience while maintaining functionality and ease of navigation.

The frontend of the project is built using React.js, while the backend is powered by Node.js and Express.js.

To run the backend server, open a terminal and execute:

```terminal
cd server
node server.js
```

To start the frontend, open another terminal and execute:

```terminal
cd client
npm start
```

Foodie is built as a recipe exploration platform that blends AI interaction with culinary discovery, allowing users to learn, explore, and interact with food in a more personalized way.
